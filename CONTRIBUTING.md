# Contributing to the Frontend

We welcome contributions to the UI!

## Development Setup
1.  Ensure the Backend is running.
2.  Install dependencies: `npm install`.
3.  Start dev server: `npm run dev`.

## Pull Request Guidelines
- **Branch Naming**: `feat/component-name` or `fix/visual-bug`.
- **Code Style**: Please follow the [Style Guide](./STYLE_GUIDE.md).
- **Responsive Check**: Ensure your changes look good on Mobile, Tablet, and Desktop.

## Linting
Run `npm run lint` before opening a PR to catch any syntax or style errors.
