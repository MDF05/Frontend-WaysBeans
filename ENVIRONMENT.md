# Environment Variables

The frontend application requires the following environment variables.

## Setup
Create a `.env` file in the root of `Frontend-WaysBeans`.

## Variable Reference

| Variable | Description | Required | Example |
| :--- | :--- | :--- | :--- |
| `VITE_API_BASE_URL` | The URL of the Backend API. | Yes | `http://localhost:5000/api/v1` |

> **Note**: In Vite, env variables must be prefixed with `VITE_` to be exposed to the client-side code.
